Ми розглядали три команди, як завантажити файл у репризиторій GitHub за допомогою терміналу:

  • git add .
  • git commit -m "...."
  • git push

Також на попередньому уроці розглядали команди для налаштування системи контролю версій git, та використовували такі команди:

  • git —version
  • ssh-keygen -t ed25519 -C "емейл"
  • git config --global user.name "John Doe"
  • git config --global user.email johndoe@example.com
  • eval "$(ssh-agent -s)"
  • ssh-add ~/.ssh/id_ed25519
  • cat ~/.ssh/id_rsa.pub
